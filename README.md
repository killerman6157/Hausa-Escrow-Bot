# Hausa Escrow Bot 🤖

Wannan Telegram bot ne da ke sauƙaƙa gudanar da amanar ciniki (escrow) tsakanin mai siye da mai sayarwa.

## Features
- 🛡️ Escrow deal flow
- 🔐 Admin confirmation
- ✅ Buyer confirmation before fund release
- 🚫 Ban functionality
- 📥 Logs to a private channel
